Les SDKs encapsulent l'authentification, les retries et la validation des payloads.

## Langages cibles

- TypeScript
- Go
- Python

<Callout title="Versioning">
Les SDKs suivent le versioning semantique. Une breaking change doit bump la version majeure.
</Callout>
