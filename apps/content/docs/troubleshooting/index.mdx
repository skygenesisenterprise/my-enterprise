Cette section regroupe des pistes de diagnostic pour les erreurs frequentes.

## Methode

1. Reproduire avec un `request_id`.
2. Verifier les logs du produit concerne.
3. Comparer l'environnement et la configuration.
4. Escalader avec les elements collectes.
