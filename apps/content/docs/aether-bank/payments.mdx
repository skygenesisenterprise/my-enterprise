<APIEndpoint method="POST" path="/v1/payments" description="Cree une intention de paiement." />

Un paiement passe par plusieurs etats : `created`, `authorized`, `captured`, `failed`, `refunded`.

## Exemple

```ts
await fetch("/v1/payments", {
  method: "POST",
  headers: { "content-type": "application/json" },
  body: JSON.stringify({
    amount: 4900,
    currency: "EUR",
    reference: "order_123",
  }),
});
```
