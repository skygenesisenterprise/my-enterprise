Aether Identity fournit les primitives d'identite pour les applications SGE : tenants, utilisateurs, clients OAuth2, sessions, MFA et federation.

## Capacites

- authentification centralisee
- OAuth2 Authorization Code + PKCE
- OpenID Connect
- clients machine-to-machine
- sessions web et mobile
- multi-factor authentication

## Quand l'utiliser

Utilise Aether Identity pour proteger une application, emettre des tokens, federer une organisation ou standardiser les permissions entre produits.
