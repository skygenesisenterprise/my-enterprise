Les sessions Aether Identity sont concues pour separer l'experience utilisateur de la duree de vie des access tokens.

## Durees recommandees

| Element | Duree indicative |
| --- | --- |
| Access token | 5 a 15 minutes |
| Refresh token | 7 a 30 jours |
| Session web | selon politique tenant |
