---
title: Errors
description: Comprendre les erreurs API.
---

Les erreurs sont structurees avec un code stable, un message lisible et un `request_id`.

| Code | Signification |
| --- | --- |
| `invalid_request` | Payload invalide |
| `unauthorized` | Token absent ou invalide |
| `forbidden` | Permissions insuffisantes |
| `rate_limited` | Trop de requetes |
